This repository contains the main analysis code used for my MSci project, focussed on heating and ventilation system performance within the UCL Estate.

## Files
- 'ct_analysis.ipynb'
Analysis of the CT circuit, including flow rate, temperature difference, and thermal power behaviour. 

- 'aranet_analysis.ipynb'
Analysis of the Aranet indoor environmental data, including variables like room temperature and CO2 concentration.

- 'funcs.py'
Helper functions created for the analysis for data cleaning and filtering.

- CSV Files
The CSV files included are the datasets used in the analysis. They are provided for transparency and reproducability of the code.
